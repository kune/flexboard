story #42 ● In Progress

Add comment feature to cards

Description

Background

As a user I want to be able to add comments to cards so that team members can discuss the task inline without leaving the board. Currently all discussion happens in external channels (Slack, email) and context is lost.

Scope

  • Comment input field at the bottom of the card detail view
  • Comments support Markdown formatting
  • Each comment shows author avatar, name, and timestamp
  • Comments can be edited and deleted by their author
  • Activity log distinguishes between comments and system events

Notes

Refer to Figma design v3 for interaction details. Mobile layout still TBD. Backend endpoint: POST /api/v1/cards/:id/comments

Acceptance Criteria
  • Comment input field visible on card detail
  • Markdown is rendered correctly in comments
  • Timestamps shown in local timezone
  • Author can edit their own comment within 15 minutes
  • Author can delete their own comment
  • Activity log entry created on new comment
